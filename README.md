# Calculator Project (C#)

This repository contains a simple calculator project written in C#.

It includes different versions of console-based calculators, from basic arithmetic to expression evaluation.

---

## Features

### Calculator v1
- Supports basic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Allows multiple number inputs
- Console-based interface

### Calculator v2
- Accepts full mathematical expressions (e.g. 2+3*5)
- Uses DataTable.Compute for evaluation
- Loop system for continuous usage
- Simple command-line interface

---

## Technologies Used
- C#
- .NET Console Application
- System.Data (for v2 expression evaluation)
